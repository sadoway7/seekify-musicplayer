    Soulseek (slskd)

    Downloads require a shared completed-downloads folder. Turn on Soulseek in Search Sources above, then add slskd credentials here. MusicGrabber can only import Soulseek downloads if slskd's completed downloads directory is mounted into this container and entered below. Use the path as MusicGrabber sees it, for example /downloads.
    The path slskd writes completed downloads to, as seen from MusicGrabber's container. This is usually a Docker volume mount, not the host path and not necessarily slskd's /downloads path.

    Monochrome (Qobuz / Tidal)

    Enable Monochrome in Search Sources above to get lossless FLAC from Qobuz via the Tidal catalogue. No account or API key required. The default public proxy is shared infrastructure, so be reasonable with it. You can point these at a self-hosted hifi-api + Qobuz proxy if you prefer.
    Tidal metadata proxy (hifi-api). Used for search and Tidal playlist import.
    Qobuz stream resolver. Fetches the actual FLAC CDN URL given an ISRC.
    Hidden Monochrome fallback used when all configured Qobuz proxies fail.

    User Management

    Admin — full run of the place. Can change every setting, manage users, and see Stats.

    User — their own per-user settings (Navidrome / Jellyfin / Lidarr creds, Spotify cookies, notifications, subfolders, change password). System-level settings stay admin-only. Stats tab visible.

    Peon — locked-down request-only account. Search, Bulk Import, Queue and Watched, that is the lot. No Settings, no Stats, no Albums, and the Convert toggle is hidden too: they always download in whatever format the admin has chosen globally. Ideal for the family member who keeps changing the audio format to MP3.

    Heads up: Creating the first account enables multi-user mode. A login screen will appear from this point on. This account will be the admin and inherit all current settings. There is no undo.
